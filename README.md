# waletz | Money without borders

**waletz** is a premium, high-converting global capital infrastructure platform designed for the modern economy. It provides a seamless experience for managing daily financial lives across 170+ countries with zero borders.

## Key Features

- **Global Capital Infrastructure**: Built for global settlement coverage and enterprise-grade liquidity.
- **Data with a Vision**: Uses proprietary AI to predict cash flow gaps, identify fraudulent patterns, and surface growth opportunities.
- **Premium Design**: A stunning, high-contrast UI with glassmorphism, bold typography (font-gotham), and smooth micro-animations.
- **Enterprise-Grade Security**: SOC2 Type II, AES-256 protocol, and GDPR compliant.
- **Developer First**: Robust SDKs and APIs for seamless integration.

## Technologies Used

- **Frontend**: React, TypeScript, Tailwind CSS
- **Icons**: Lucide React
- **Animation**: CSS Keyframes & Tailwind Utilities
- **AI Integration**: waletz Intelligence API

## Run Locally

**Prerequisites:** Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
